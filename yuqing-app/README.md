# 舆情分析助手 UI 原型

基于 Taro 3 + Taro UI 的多端原型，支持 **H5** 与 **小程序**（微信/支付宝/百度）。

## 快速开始

```bash
npm install --legacy-peer-deps   # 若报错请使用此命令
npm run dev:h5                   # Web 开发（浏览器访问）
npm run dev:weapp                # 微信小程序（用开发者工具打开 dist/weapp）
```

## 页面结构

- **首页**：搜索框、新闻热点（跑马灯 + 列表）、历史搜索
- **结果页**：标签 1 新闻汇总、标签 2 思考过程（子标签 + 流式内容）、标签 3 分析结果（文字 + 进度/图表）
- **我的**：用户信息、菜单（历史、收藏、通知、设置）

详细说明见项目根目录 `docs/开发文档-舆情分析助手UI原型.md`。
